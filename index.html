캘린더 AI

  • F2 아젠다 초안 생성 — 이전 회의록 참고, 시간 배분 포함 자동 작성 P1
  • F3 회의록 자동 생성 — 트랜스크립트 기반, 결정사항·발화자별 구조화 P0
  • F4 후속 태스크 연동 — 액션 아이템 파싱, WORKS 게시판 일괄 등록 P2
